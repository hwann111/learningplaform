## Learningplatform
학습 도구 제작 웹페이지입니다.
생성형 AI를 활용하여 학습자에게 학습도구들을 제공합니다.
제공되는 학습 자료는 동영상, 맞춤 질문 제작 그리고 수준별 설명 기능이 있습니다.
